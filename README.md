# Seguimiento TDAH

App web gratuita para el seguimiento diario de personas con TDAH:
agenda por horas, registro de sueño, cierre y balance del día, bloc de
notas y una herramienta de valores → objetivos → acciones.

Diseñada por el **Dr. Jordi Risco**, psiquiatra y psicoterapeuta, como
herramienta de autoseguimiento personal — no sustituye una valoración,
diagnóstico ni tratamiento profesional.

Todo lo que anota cada persona se guarda únicamente en su propio
navegador (no hay servidor ni base de datos) — nada se envía ni se
almacena en ningún sitio fuera de su dispositivo.

**App:** https://jordirisco.github.io/seguimiento-tdah/
**Consulta:** https://calendly.com/jordiriscopsiquiatria/consulta-online
