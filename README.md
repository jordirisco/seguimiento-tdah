# Seguimiento TDAH

Prototipo de app web para el seguimiento diario de pacientes con TDAH:
agenda por horas, registro de sueño, cierre y balance del día, bloc de
notas y una herramienta de valores → objetivos → acciones.

Todo lo que anota cada persona se guarda únicamente en su propio
navegador (no hay servidor ni base de datos) — nada se envía ni se
almacena en ningún sitio fuera de su dispositivo.

**Demo:** https://jordirisco.github.io/seguimiento-tdah/
